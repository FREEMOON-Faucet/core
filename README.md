# The FREEMOON Faucet - Money on Tap

## Quick Setup

To clone repository and install dependencies, in a terminal, run:

```bash
git clone https://github.com/paddyc1/FREEMOON-Faucet.git
cd FREEMOON-Faucet
npm install
```

## Tests

To compile and view test results, run:

```bash
npx hardhat compile
npx hardhat test
```
